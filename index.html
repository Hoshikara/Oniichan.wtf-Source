Different Modes

The mode you selected will be saved on board.


  • Keyboard mode: Knobs will work as mouse input, while buttons output as keys.

    • Hold START + BT-A down together WHILE plugging the controller to the PC.


  • Controller + Mouse: Controller inputs as a gamepad, while knobs will work as mouse input.

    • Hold START + BT-B down together WHILE plugging the controller to the PC.


  • Controller HID mode: Controller inputs as a gamepad.

    Hold START + BT-C down together WHILE plugging the controller to the PC.


  • Download Mode: Lights up all buttons and is used to download the firmware update.

    Hold FX-L + FX-R down together WHILE plugging the controller to the PC.


Yuancon Setup


Unnamed SDVX Clone


  1. Use Mode 3.

  2. Head to the Settings menu.

  3. Set Button input mode and Laser input mode to Controller and set Selected Controller to Yuancon.

  4. Now head to the top where it says Input and map your controller.

  5. You can now adjust the Laser sensitivity to your liking. However, it's recommended to use a sensitivity of 1.875.

  6. If you want a more "cab feel" head to Input and look for Laser Assist. Afterward, change the value in Base Laser Assist to 1.1 or 1.2.

  7. If you're hitting notes too early or late, then head to Input Offset and use the Offset Calibration to help adjust.

  8. Done. Have fun!


K-Shoot Mania


  1. Use Mode 3.

  2. Go to Option, then Key Configuration.

  3. On the bottom where it says "Keyboard 1", press right twice to change it to "Gamepad 1"

  4. Map your buttons using the arrow keys and your controller. (you won't be able to map knobs which is normal)

  5. Press Esc, then go to Input / Judgement settings.

  6. Scroll down to Input Type of Laser Object, then select Analog Stick X/Y.

  7. Adjust knob sensitivity at Slider/Mouse signal sensitivity.

  8. Restart the game.

  9. Done. Have fun!


source: Jeremii#0418 on Discord


SOUND VOLTEX


  1. Use Mode 3 but if that doesn't work then you can use this program.

  2. You can also use Mode 1 with a sensitivity of 0.5-0.8 in Spice Tools if you're not using EAC.

Frequently Asked Questions


My controller didn't include an Allen key nor screwdriver. What should I do?
The controller uses a 2.5mm Hex Allen key to unscrew the knobs and case. If you have a v1 Yuancon, you'd need a size #2 screwdriver to open the case.
I ordered a Sound Voltex Controller, and it's stuck on "payment received." How long will it take to be shipped?
It takes one week to build the controller and 5 to 7 days to be delivered, depending on the country.
Is there a way to replace the paper inside the buttons?
Yes, you can either print or cut the paper in the following dimensions:
  • BT: 4.6x4.6cm
  • FX: 3.7x1.9cm
  • START: 2.1x2.1cm

After you've printed or cut the paper, you'll have to disassemble the controller now.

What's the difference between CMOD, XMOD, and MMOD?
CMOD: Constant Hi-Speed with no changes. (cheater mod)
XMOD: Multiply by BPM. (default sdvx hispeed behavior)
MMOD: Finely tuned speed that can be locked in to persist between songs. (sdvx floating hispeed behavior with double tap start/red number when locked in)
Is there a guide on a DIY SDVX Controller?
Yes, there's multiple guides on how to build your own SDVX Controller. There's an SDVX Controller & Pocket SDVX guide.
My Controller has no input. What should I do?
Try using a USB 2.0/3.0 port and make sure if you're using a USB HUB that it has enough power to run the controller. If that doesn't solve the issue, open the case and check if there's anything unplugged.
Is there a way to make the Yuancon buttons brighter? It's dim for me.
To make the button lights brighter you can do so by disconnecting the RGB lights. Another way to do so is to short all the resistors marked "RL" but that might void your warranty.
Does Yuancon support Linux?

Yuancon doesn't support Linux officially, but it can work on Linux with a few steps. There's two methods currently available.

    Method 1:
    Requirements:
    • Xorg display server
    • The controller must be in Mode 1

    1. Install evdev
    2. Create a file named 10-yuancon.conf in /etc/X11/xorg.conf.d/ containing these lines: 
      
Section "InputClass" 
    Identifier "Yuancon"
    MatchProduct "YuanCon YuanCon Mouse"
    MatchDevicePath "/dev/input/event*"
    Driver "evdev"
EndSection
    3. Restart the X server or your PC

      4. source: Jeremii#0418 on Discord



    Method 2:
      Requirements:
      • Xorg display server
      • The controller must be in Mode 3

      1. Install libinput
      2. Create a file named 10-yuancon.conf in /etc/X11/xorg.conf.d/ containing these lines: 
        
Section "InputClass"
    Identifier "Yuancon"
    MatchProduct "YuanCon YuanCon Mouse"
    MatchDevicePath "/dev/input/event*"
    Driver "libinput"
EndSection
      3. Restart the X server or your PC

What kind of lubricant do I use for my buttons/knobs?
You can use Silicone or Lithium based lubricant. DO NOT USE VASELINE OR WD-40.
